GET 요청 예

1) 웹 브라우저 주소창에 URL을 입력했을 때

2) 다음과 같이 링크를 클릭했을 때

servlet15

href="상대경로" : 현재 URL을 기준으로 서버에 요청한다.
(상대경로란? /로 시작하지 않는 경로를 의미한다.)
href="servlet15" :
현재 URL이 http://localhost:8080/web02/step04/getpost.html
이기 때문에 요청 URL은 현재 URL을 기준으로 다음과 같다.
http://localhost:8080/web02/step04/servlet15

3) 입력 폼의 submit 버튼을 클릭했을 때

action="상대경로" : 위에서 설명한 상대 경로 규칙과 같다.
